# MRI Simulator

**Interactive MRI scan planning simulator for radiographers and radiography students.**

---

## ▶ [Open the Simulator →](https://suhaibshd7.github.io/mri-simulator/)

---

### What it covers

* Brain, IAM/IOM, Spine, Knee, and more anatomy regions
* Multi-plane scout planning — Sagittal, Coronal, Axial
* Coverage box drawing with rotation angle control
* Scored feedback: angle accuracy, coverage quality, position accuracy
* Calibration mode for instructors to build and customise exam content
* Export / Import full configuration as `.json` for sharing or backup
* In-app planning guide with scan rules per anatomy region

### How to use on mobile

1. Open the link above in **Chrome** (Android) or **Safari** (iOS)
2. Tap **⋮ menu → Add to Home Screen**
3. The app installs like a native app — works offline

### No download required

The simulator runs entirely in the browser. No account, no install, no cost.
